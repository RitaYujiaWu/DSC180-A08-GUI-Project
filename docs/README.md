# DSC180 A08 — Project Website (Q2)

Static website for **Inference-Time Scaling for GUI Agents with Process Reward Models and Internal World Models** (DSC180 A08 Capstone, Quarter 2). The site is served from the `/docs` folder (e.g. GitHub Pages “Deploy from a branch” → source: **docs**).

## Link to website

- **Live site (GitHub Pages):**  
  When GitHub Pages is enabled for this repo with source **docs**, the site is available at:
  `https://<username>.github.io/DSC180-A08-GUI-Project-1/`

- **Local preview:**  
  ```bash
  cd docs && python3 -m http.server 8000
  ```
  Then visit `http://localhost:8000`.

## Contents

- **Introduction** — Motivation, GUI agents, inference-time scaling (PRM + world model).
- **Methods** — Part A (PRM pipeline and Android Lab case study), Part B (internal world model).
- **Results** — PRM evaluation (OSWorld), PRM-guided agent benchmarks, World Model evaluation (WebVoyager).
- **Conclusion** — Summary and impact.
