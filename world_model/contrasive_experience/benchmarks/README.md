# Benchmarks

Evaluation modules for different GUI agent benchmarks.

## Benchmarks

- **`webvoyager_evaluation/`** - WebVoyager benchmark evaluation
- **`MMInA_evaluation/`** - MMInA benchmark evaluation  
- **`Mind2Web_evaluation/`** - Mind2Web benchmark evaluation
- **`mmina/`** - MMInA dataset and utilities

Each benchmark has its own `test_runner.py` and `evaluator.py` for running evaluations.
